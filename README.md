#  FullStack Microservices Platform

A production-style multi-language microservices system demonstrating scalable backend architecture, communication patterns, and distributed systems design.



##  Overview

This project demonstrates how to build a **modern distributed backend system** using microservices. Each service is independently developed, deployed, and scaled, following best practices used in real-world systems.



##  Tech Stack

### Languages & Frameworks
- Java 17 (Spring Boot)
- Node.js (TypeScript)
- Python (FastAPI)
- Go

### Databases
- MariaDB (Relational DB)
- Redis (Caching / In-memory DB)
- ClickHouse (Analytics)

### Messaging & Communication
- REST APIs (HTTP)
- WebSockets (real-time communication)
- Apache Kafka (event-driven messaging)

### Infrastructure
- Docker & Docker Compose
- Nginx (Reverse Proxy)
- HAProxy (Load Balancer)

### Monitoring (Ready Setup)
- Grafana
- Kibana
- Telegraf
